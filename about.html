Sobre mim

Tenho habilidades em Golang, Python, C/C++, HTML, CSS e PostgreSQL, e já atuei em projetos que envolvem desenvolvimento web, mobile e sistemas embarcados. Se quiser trocar uma ideia ou conhecer mais sobre meu trabalho, fique à vontade para entrar em contato!

Tenho alguns projetos de diversas linguagens de programação em meu github que envolvem areas diveras, como, desenvolvimento de Dashboads voltados para probabilidade e estatistica, software voltado para datos taxonamicos, dentre outros projetos.
