Dashboard de probabilidade e estatística sobre cyberataques

O banco de dados utilizado para a montagem deste dashboard foi Incidentes de Segurança da Informação no Brasil.

O dashboard tem como objetivo resumir os ataques cibernéticos que ocorreram no período de 2010 a 2019 no Brasil, mostrando a frequência e em que mês do ano eles ocorrem, podendo filtrar os ataques em: Worm, DOS, Invasão, Web, Scan, Fraude e Outros.

Projeto Cadmus

O Cadmus é um projeto desenvolvido em Django que funciona como um banco de dados taxonômico. Ele foi projetado para armazenar, organizar e gerenciar informações sobre diferentes classificações de organismos, permitindo consultas eficientes e estruturadas. A plataforma possibilita a categorização hierárquica dos dados, seguindo padrões taxonômicos, e pode incluir informações detalhadas sobre espécies, gêneros, famílias e outros níveis de classificação. Além disso, o sistema pode oferecer recursos como busca avançada, filtros personalizados e integração com outras bases de dados científicas. O objetivo do Cadmus é facilitar o acesso e a organização de dados biológicos, tornando-se uma ferramenta útil para pesquisadores, estudantes e profissionais da área.

Link para o repositório no GitHub: GitHub Repository.
